## 映射介绍

映射用来保存 key-value 数据，可以使用 map 关键字来创建映射。

## 定义映射的几种方式

```
// 申明一个映射变量，这时映射并有没真正被创建
var a map[string]int
// 真正创建一个映射
a = make(map[string]int)

// 定义一个映射
b := make(map[string]int)

// 定义一个带有键值数据的映射
c := map[string]int{"a":1, "b":2}
```

## 零值

映射的零值是 nil，和切片一样。
